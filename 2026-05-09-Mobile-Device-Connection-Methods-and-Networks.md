## 📚 4. Steps Completed  
### 📱 Mobile Device Connection Methods  
I reviewed the core connectivity methods used by mobile devices:

- **USB types:** USB‑C, MicroUSB, MiniUSB  
- **Lightning:** Apple’s proprietary connector  
- **Wi‑Fi Direct:** High‑speed peer‑to‑peer wireless  
- **Hotspot:** Phone acting as a portable WAP to create a WLAN  
- **Tethering:** Sharing mobile data via USB, Bluetooth, or Wi‑Fi  
- **Bluetooth pairing:** Enable → pair → PIN → test connectivity  
- **NFC:** Short‑range, secure communication for payments and authentication  

I reinforced that **USB tethering is faster and more stable** than hotspot due to reduced interference and direct wired connection.

### 📡 Mobile Device Networks (2G → 5G, GSM, CDMA)  
I studied the evolution of mobile networks:

- **2G:** Digital voice + SMS, limited data  
- **3G:** Mobile internet, video calls, GPS support  
- **4G/LTE:** Broadband‑level speeds (150–300 Mbps), global standard  
- **5G:** 100 Mbps–10 Gbps, ultra‑low latency, IoT‑focused  

I also reviewed GSM vs CDMA, PRL behavior, SIM/eSIM functionality, Wi‑Fi calling, hotspot behavior, and cellular vs Wi‑Fi data usage.

## 🧠 6. What I Learned  
- USB tethering is faster and more stable than hotspot  
- GSM uses SIM cards; CDMA uses PRL  
- 5G provides low latency and high throughput for IoT  
- MDM controls the entire device; MAM controls only corporate apps  
- Digitizer handles touch input; LCD handles image display  
- SODIMM = laptop RAM; DIMM = desktop RAM  
- ECC RAM is used in servers for error correction  
- Data caps must be considered when syncing over cellular
