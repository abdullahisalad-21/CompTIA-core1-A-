## 3️⃣ 🧠 Key Concepts I Learned

### LCD Family (Requires Backlight)
- IPS  
- TN  
- VA  
- LED‑backlit LCD  
- Mini‑LED  
- QLED  
- CCFL LCD  
- Edge‑lit / Full‑array LCD  

**Traits:** Backlight required, no burn‑in, liquid crystals, can have IPS glow/backlight bleed.

### OLED Family (Self‑Emissive)
- OLED  
- AMOLED  
- QD‑OLED  
- WOLED  
- Micro‑OLED  

**Traits:** No backlight, perfect blacks, infinite contrast, ultra‑thin, burn‑in risk.

### Panel Characteristics
- IPS: High color, wide angles, slower response  
- TN: Fastest response, low color, poor angles  
- VA: Best contrast, good color, decent angles  

### Mini‑LED
- Still LCD  
- Uses tiny LEDs for backlight  
- High brightness  
- Excellent HDR  
- Not self‑emissive  

### Digitizer
- Input component  
- Converts analog → digital  
- Enables stylus/finger control  

### Inverter
- Converts DC → AC  
- Powers CCFL backlights  
- Not used in OLED  

### Burn‑In Rule
Self‑emissive = burn‑in (OLED).  
Backlight = no burn‑in (LCD).

## 7️⃣ 📚 Resources & Memory Tricks Added
**Backlight Rule**  
If it has a backlight → LCD.  
If each pixel lights itself → OLED.

**Burn‑In Rule**  
Only OLED can burn in.  
LCD cannot.

**Panel Summary**  
- IPS = Color + Angles  
- TN = Speed  
- VA = Contrast  
- OLED = Perfect blacks
